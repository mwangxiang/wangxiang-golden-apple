# Workflow

1. Export WeFlow messages into JSON in the private production workspace.
2. Build persona data and avatar reference data.
3. Generate an image prompt pack.
4. Use the image generation model for final visual posters.
5. Copy final PNGs into the run `generated/` folder and the D drive downloads folder.
6. Run `validate-run.mjs`.
7. Copy selected final artifacts into this private repository under `private-results/`.

This repository is for storing the reusable method and selected results. It is not the live WeFlow data workspace.
