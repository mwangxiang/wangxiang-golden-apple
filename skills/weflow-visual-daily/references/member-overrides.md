# Member Overrides

Use this file when a generated member portrait is visually wrong, too aggressive, or based on a placeholder avatar.

## 风中梧桐

Preferred direction:

- gentle mentor
- silver mask as a calm symbolic accessory
- warm, restrained expression
- thoughtful community guide
- elegant, non-combat posture

Avoid:

- mecha
- Ultraman-like armor
- battle suit
- hard-core sci-fi
- weapon-like props
- intimidating pose

Prompt delta:

```text
Make 风中梧桐 a warm silver-mask mentor with soft classroom lighting, elegant calm posture, and no mecha or combat elements.
```

## Placeholder avatar fallback

If `avatarStatus=placeholder`, do not describe the placeholder image. Generate a replacement from:

- group nickname
- known role
- representative quote
- user-provided reference
- manually approved visual direction
